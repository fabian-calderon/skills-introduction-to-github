# Guía de contribución

Gracias por tu interés en contribuir a este proyecto.

## Cómo contribuir

1. Haz un fork del repositorio.
2. Crea una rama para tu cambio.
3. Realiza tus modificaciones.
4. Haz commit con un mensaje claro.
5. Envía tus cambios a tu fork.
6. Abre un Pull Request hacia el repositorio original.

## Recomendaciones

- Mantén los cambios pequeños y enfocados.
- Explica claramente qué problema resuelves.
- Si la mejora es grande, abre primero un Issue para discutirla.
- Revisa el README antes de contribuir.

## Estilo

- Usa nombres claros.
- Mantén la documentación simple.
- Evita cambios innecesarios fuera del alcance del PR.
