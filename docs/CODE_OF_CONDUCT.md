# Código de conducta

Este proyecto busca mantener una comunidad abierta, respetuosa y profesional.

## Comportamiento esperado

- Tratar a los demás con respeto.
- Aceptar comentarios técnicos de forma constructiva.
- Mantener discusiones enfocadas en el proyecto.
- Colaborar de manera profesional.

## Comportamiento no aceptable

- Lenguaje ofensivo o discriminatorio.
- Ataques personales.
- Conductas hostiles o irrespetuosas.
- Publicación de contenido fuera del contexto del proyecto.

## Aplicación

Los mantenedores del proyecto pueden moderar conversaciones, solicitar correcciones o rechazar contribuciones que no respeten estas normas.
